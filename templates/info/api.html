How to use the RDSL API

Add/remove repls to RDSL Pinger API

ADD a repl via the Replit Shell via POST requests

Run this in a Replit Shell to ADD the URL to the pinger!
curl https://up.rdsl.ga/api/cli -X POST -F "add=https://${REPL_SLUG}.${REPL_OWNER}.repl.co"

REMOVE a repl via the Replit Shell

Run this in a Replit Shell to REMOVE it from the pinger via POST requests
curl https://up.rdsl.ga/api/cli -X POST -F "add=rem https://${REPL_SLUG}.${REPL_OWNER}.repl.co"

Or use an external source with a POST request

Bash example: It should look like this:
curl https://up.rdsl.ga/api/cli/json -X POST -F "add=https://repl.guy.repl.co"

import fetch, { FormData } from 'node-fetch';

Python POST example:

Python should look like this:

    'add': (None, 'https://repl.guy.repl.co'),

The https://up.rdsl.ga/api/cli/json URL returns data in JSON via the argument msg.


Stats API

Small notice: This has a ratelimit of 1 request per 30 seconds!

How to get stats from RDSL Pinger:

Using Bash it would look like this:
curl https://up.rdsl.ga/api/stats

Then you get a JSON response with all stats

Python example GET all stats data

Python should look like this

f = request("https://up.rdsl.ga/api/stats")
print(f)

Then you also get a JSON response with all stats
