Other pingers

Sometimes, Replit doesn't persist files. This means that now and then, when you add your webserver, it will stop being pinged after a few hours. If this happens to you, try again for a bit or use these other pingers:

REMOVE a repl via the Replit Shell

Run this in a Replit Shell to REMOVE it from the pinger!
curl https://up.rdsl.ga/api/cli -X POST -F "add=rem https://${REPL_SLUG}.${REPL_OWNER}.repl.co"

Or use an external source with a POST request

It should look like this:
curl https://up.rdsl.ga/api/cli -X POST -F "add=https://repl.guy.repl.co"
